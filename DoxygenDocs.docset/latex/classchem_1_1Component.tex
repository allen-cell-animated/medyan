\subsection{Detailed Description}
\hyperlink{classchem_1_1Component}{Component} class is the base class for the \hyperlink{classchem_1_1Composite}{Composite} pattern hieararchy. 

The \hyperlink{classchem_1_1Composite}{Composite} patter allows building of complex hieararchical objects, with convenient methods for applying a function to all nodes (i.\-e. the \hyperlink{classchem_1_1Visitor}{Visitor} pattern). Each node in the hieararchy may have a parent and may contain several children nodes. A class that is derived directly from \hyperlink{classchem_1_1Component}{Component} and not from \hyperlink{classchem_1_1Composite}{Composite}, cannot contain children, i.\-e. it is a leaf node. \begin{DoxyNote}{Note}
\hyperlink{classchem_1_1Component}{Component} objects may contain \hyperlink{classchem_1_1Species}{Species} and \hyperlink{classchem_1_1ReactionBase}{Reaction\-Base} collections, however, this is treated seperately from the \hyperlink{classchem_1_1Composite}{Composite} pattern (i.\-e. separate methods exist for the corresponding access to elements, changes, etc.) 
\end{DoxyNote}
