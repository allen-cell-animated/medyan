\subsection{Detailed Description}
\hyperlink{classchem_1_1Composite}{Composite} class is the aggregating class for the \hyperlink{classchem_1_1Composite}{Composite} pattern. 

The \hyperlink{classchem_1_1Composite}{Composite} pattern allows building of complex hieararchical objects, with convenient methods for applying a function to all nodes (i.\-e. the \hyperlink{classchem_1_1Visitor}{Visitor} pattern). Each node in the hieararchy may have a parent and may contain several children nodes. A class that is derived from \hyperlink{classchem_1_1Composite}{Composite} can contain children \hyperlink{classchem_1_1Component}{Component} objects. \begin{DoxyNote}{Note}
\hyperlink{classchem_1_1Composite}{Composite} objects may contain \hyperlink{classchem_1_1Species}{Species} and \hyperlink{classchem_1_1ReactionBase}{Reaction\-Base} collections, however, this is treated seperately from the \hyperlink{classchem_1_1Composite}{Composite} pattern (i.\-e. separate methods exist for the corresponding access to elements, changes, etc.) 
\end{DoxyNote}
