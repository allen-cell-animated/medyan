\subsection{Detailed Description}
An abstract interface to traverse those nodes in the \hyperlink{classchem_1_1Composite}{Composite} pattern which fulfil a certain predicate. 

The visitor pattern allows a functor to visit each node of the \hyperlink{classchem_1_1Composite}{Composite} pattern. Concrete classes derived from \hyperlink{classchem_1_1Visitor}{Visitor} must implement the \hyperlink{classchem_1_1Visitor_a710015ef735e109c812edbb653d6d598}{visit(\-Component $\ast$c)} method. \hyperlink{classchem_1_1Visitor}{Visitor}, v, allows a predicate to be applied at each node, $\ast$c, and cv-\/$>$visit(c) is called only if that node satisfies the predicate. For example, this can be used to visit only filaments in the hieararchy.
