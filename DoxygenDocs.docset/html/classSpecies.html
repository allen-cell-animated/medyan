Detailed Description

Species class represents chemical molecules, tracks their copy number and can be used in Reactions.

This class represents chemical species, such as G-Actin. Tracks the copy number of molecules and the Reactions in which it is involed (

See also:
Reaction).
Note:
In the future may implement the Logging and Persistence interfaces.
Each intantiation of Species is unique, and hence, cannot be copied.
