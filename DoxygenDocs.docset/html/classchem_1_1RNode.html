virtual void chem::RNode::activateReaction ( ) [pure virtual]

This method is called by Reaction::activateReaction().

Its effect depends on the underlying stochastic simulatio algorithm. For example, in the NRM algorithm, a new tau and a are computed and the heap is updated.

virtual void chem::RNode::passivateReaction ( ) [pure virtual]

This method is called by Reaction::passivateReaction().

Its effect depends on the underlying stochastic simulatio algorithm. For example, in the NRM algorithm, a tau is set to infinity and the heap is updated.
