Detailed Description

SpeciesType class represents the type of chemical Species (for example, a Species with name "Arp2/3" and SType Diffusing)

This class describes the type of Species. The description is based on a name (std::string) and SType (Bulk, Diffusing, etc.).

Note:
In the future may implement the Logging and Persistence interfaces.
SpeciesType is used by Species with the flyweight pattern, since in a large system millions of unique Species may exist, but only a handful of SpeciesType objects. For example: one ("F-Actin",Filament) may correspond to millions of individual Species.
