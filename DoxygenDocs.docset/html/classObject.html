Detailed Description

This is the base class of Species, Reactions, Compartments, Filaments, etc.

Note:
A class derived from Object cannot be copied, but instead should be duplicated in a special way. Classes inherited from Object need to implement the Persistence interface, so to be able to write themselves into an output filestream and read themselves from an input filestream
