A boundary shape that holds Boundary\+Elements. 

The \hyperlink{classBoundarySurface}{Boundary\+Surface} class is a basic boundary shape that is owned and controlled by the \hyperlink{classBoundary}{Boundary}\mbox{]} that created it. It holds a vector of \mbox{[}Boundary\+Elements\mbox{]}(\hyperlink{classBoundaryElement}{Boundary} as well as any other geometrical information needed for the given implementation.
