\subsection{Detailed Description}
An abstract base class for algorithms that run stochastic chemical kinetics. 

Specific stochastic kinetics algorithm classes should inherit from \hyperlink{classChemSimImpl}{Chem\+Sim\+Impl}. A user will then attach the corresponding algorithm to \hyperlink{classChemSim}{Chem\+Sim} via the algoritm's base class \hyperlink{classChemSimImpl}{Chem\+Sim\+Impl}.
