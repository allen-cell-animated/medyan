Detailed Description

A container to store a MBranchingPoint and CBranchingPoint.

BranchingPoint class is used to manage and store a MBranchingPoint and CBranchingPoint. Upon intialization, both of these components are created.

Extending the Movable class, the positions of all instances can be updated by the SubSystem.
