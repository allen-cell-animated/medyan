Detailed Description

Represents a single coordinate between Cylinders, and holds forces needed for mechanical equilibration.

Beads are the "hinges" between Cylinders. In the minimization algorithms, beads are moved corresponding to external forces, for example, Filament stretching and bending. The bead class contains currernt coordinates and forces, and has functions to calculate dot products for the minimization algorithms.

Extending the Movable class, the positions of all instances can be updated by the SubSystem.

Extending the DynamicNeighbor class, all instances can be kept in NeighborLists.
