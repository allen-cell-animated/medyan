Detailed Description

Represents an element of a boundary surface.

The BoundaryElement class is a representation of a BoundarySurface element, which can interact with other elements in the system, including other BoundaryElements as well as Beads in Filaments. Together, a collection of boundary elements make up a BoundarySurface.
