Detailed Description

An abstract base class for algorithms that run stochastic chemical kinetics.

Specific stochastic kinetics algorithm classes should inherit from ChemSimImpl. A user will then attach the corresponding algorithm to ChemSim via the algoritm's base class ChemSimImpl.
