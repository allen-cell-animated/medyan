Detailed Description

Implements the simplest version of the Gillespie algorithm, without caching, etc.

ChemSimpleGillespieImpl manages the Gillespie algorithm at the level of the network of reactions. Reaction objects can be added and removed from the ChemSimpleGillespieImpl instance.
