Detailed Description

A container to store a MCylinder and CCylinder.

Cylinder class is used to manage and store a MCylinder and CCylinder. Upon intialization, both of these components are created.

Extending the Movable class, the positions of all instances can be updated by the SubSystem.

Extending the Reactable class, the reactions associated with all instances can be updated by the SubSystem.

Extending the DynamicNeighbor class, all instances can be kept in NeighborLists.
