Detailed Description

A RSpecies implementation that changes copy number regularly.

The RSpeciesReg class represents a typical RSpecies. Upon calling up() or down(), the copy number is incremented normally. getN() and getTrueN() both return the true copy number tracked in the base class.

The RSpeciesReg can be used for any type of Species in the chemical system.
