Detailed Description

A class to represent the structure of a microtubule organizing center (MTOC)

The MTOC class provides a base structure for a mechanical MTOC, which includes a Bubble representing the MTOC and Filaments which are attached to it at the minus end by some sort of potential.

This class only has functions to set and get the constituent Bubble and Filament and is only mechanically relevant for now, but may be extended to have chemical properties in the future.
