Detailed Description

Represents a chemical object that is bound to a Filament.

The CBound class is an abstract representation of a chemically bound object to a Filament (Could be a Linker, MotorGhost, BranchingPoint, etc). Each CBound object has a pointer to the corresponding SpeciesBounds on a Filament. Different implementations of CBound class will have different functions to bind, move, etc. See documentation of subclass for more details on function.]

The CBound class also controls the element's unbinding reaction, including its creation and destruction in the chemical system.
