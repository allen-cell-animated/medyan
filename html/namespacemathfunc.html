Function to calculate a diatance between two segments.

segment 1: l1 = v2 - v1; segment 2: l2 = v3 - v4; distance betwin begining of l1 nd l2: l0 = v3 - v1; a = (l1,l1); b = (l1,l2); c = (ls,vls); d = (l1,l0); e = (l2,l0); f = (l0,l0); D = a*c - b*b;

lines are given by a parametric equation v1 + s*(v2-v1) and v3 + t*(v4-v3). Closes distance is given by l = l0 + sc*l1 - tc*l2, where tc and sc are numbers between[0,1], so points are withing the segments. Algorithm is the following: find closes points between infinite lines and then move tc and sc to 0 or 1 if they are out of [0,1].
