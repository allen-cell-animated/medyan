Detailed Description

SpeciesDB should be used for obtaining new objects of the Species hieararchy.

It internally uses the StandardSpeciesFactory to accomplish the construction. It is also a database that can be queried to obtain a pointer to a Species if the Species' ID is known. This class is a singleton, and should be used like this: 

 SpeciesDB *sdb = SpeciesDB::getInstance();
 sdb->Initialize();
