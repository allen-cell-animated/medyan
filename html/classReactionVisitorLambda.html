bool ReactionVisitor::visit ( ReactionBase *  r)
inlineinherited

When this Visitor, *v, is propagated through the Composite hieararchy, at each Component node pointer *c, the following method is called: v->visit(s), for each of the Reaction *r of that node.

If a false value is returned from this function called, then the further propagation of the tree is terminated.
