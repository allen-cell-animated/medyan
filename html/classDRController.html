Detailed Description

Used to initialize the dynamic rate components of a simulation.

The DRController intializes all dynamic rate objects in the system, as specified in the input file. The controller's intiailize function will set Cylinders, [Linkers]() and [MotorGhosts]() with corresponding static DynamicRateChanger objects.
